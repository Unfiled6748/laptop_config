Additional Steps

1. Install iTerm2
brew install --cask iterm2

2. Instal Hack Nerd Font
brew tap homebrew/cask-fonts
brew intsall --cask font-hack-nerd-font

3. Config iTerm2 to jump to beginning when you press control a
- Open iTerm2 preferences (command ,)
- Click on Keys
- Click on + 
- In Recording, press control a
- In Action, select 'Send Escape Sequence'
- In the last box, select OH 

4. Config iTerm2 to jump to end when you press control e
- Open iTerm2 preferences (command ,)
- Click on Keys
- Click on + 
- In Recording, press control e
- In Action, select 'Send Escape Sequence'
- In the last box, select OF 

5. Config Meta key in iTerm2
- Open iTerm2 preferences (command ,)
- Click on Profiles
- Click on Keys
- Go to 'Right Option Key'
- Click on Esc+
